Barebones Audio Recorder 7

Client-side audio capture and conversion to MP3.

This recorder captures user audio, converts to MP3 format on the client side, then uploads to the server.

This recorder is fully functional.

Documentation and API

Features:

  • Record, edit, play back audio in the browser window.
  • Upload MP3 file to your server.
  • All recorder functions are exposed to Javascript.
    Make your own HTML interface.

Requirements:

  • Flash player version 11.2 or higher.
  • Microphone and speakers.
